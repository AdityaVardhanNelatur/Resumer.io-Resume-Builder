# AI Resume Builder (MERN)

A full-stack AI Resume Builder built using the MERN stack.

## Features
- Authentication (Login / Signup)
- Resume Create / Edit / Delete
- Resume Upload & Parsing
- Rule-based AI Resume Improvement
- Live Preview
- PDF Download
- Dashboard UI

## Tech Stack
- React + Tailwind CSS
- Node.js + Express
- MongoDB
- JWT Authentication
